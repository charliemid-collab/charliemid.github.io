Data Types & Variables

The Foundation of Programming

Primitive Data Types
Basic building blocks of data storage
# Numbers age = 20 # Integer height = 5.9 # Float # Text name = "Alice" # String # Logic is_student = True # Boolean
Collection Types
Storing multiple values efficiently
# Ordered collections courses = ["COMP1005", "MATH1010", "PHYS1001"] student_info = ("John", "Doe", 2025) # Tuple # Key-value pairs student = { "name": "Sarah", "id": 123456, "gpa": 3.8 }
Type Conversion
Converting between different data types
# String to number user_input = "85.5" grade = float(user_input) # 85.5 # Number to string age = 20 age_str = str(age) # "20" # String operations name = "Alice Johnson" first_name = name.split()[0] # "Alice"

Key Insight: Variables are containers that hold data. The data type determines what operations you can perform!
