Recursion and Divide-and-Conquer

Part (a) - Factorial and Fibonacci (10 marks)

Implement recursive mathematical functions:

  • Function factorial(n) to calculate n! using recursion
  • Function fibonacci(n) to calculate the nth Fibonacci number
  • Include base cases and recursive cases for both functions
  • Add input validation to handle negative numbers
  • Trace the recursive calls for factorial(5) and fibonacci(6)

Show the call stack for at least one example:

factorial(5) = 5 × factorial(4) = 5 × 4 × factorial(3) = 5 × 4 × 3 × factorial(2) = 5 × 4 × 3 × 2 × factorial(1) = 5 × 4 × 3 × 2 × 1 = 120

Part (b) - Recursive List Processing (10 marks)

Implement recursive functions for list processing:

  • Function recursive_sum(numbers) to sum a list recursively
  • Function recursive_max(numbers) to find the maximum value recursively
  • Function recursive_reverse(lst) to reverse a list recursively
  • Function recursive_search(lst, target) to search for an element recursively

Each function should handle the empty list as base case

Part (c) - Merge Sort (8 marks)

Implement merge sort using divide-and-conquer:

  • Function merge_sort(arr) that recursively divides and merges the array
  • Helper function merge(left, right) to merge two sorted arrays
  • Trace the sorting process for the list [64, 34, 25, 12, 22, 11, 90]
  • Explain why merge sort has O(n log n) time complexity

Show the divide and conquer steps:

[64, 34, 25, 12, 22, 11, 90] → [64, 34, 25] + [12, 22, 11, 90] → [64] + [34, 25] + [12, 22] + [11, 90] → Merge: [25, 34, 64] + [11, 12, 22, 90] → Final: [11, 12, 22, 25, 34, 64, 90]
