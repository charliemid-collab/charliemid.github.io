Control Structures

Making Decisions in Your Code

Conditional Statements
Condition
↓
True → Action
↓
False → Alternative
# If-elif-else structure gpa = float(input("Enter GPA: ")) if gpa >= 3.7: classification = "First Class" elif gpa >= 3.0: classification = "Upper Second" elif gpa >= 2.0: classification = "Lower Second" else: classification = "Third Class" print(f"Your classification: {classification}")
Loops & Iteration
# While loop - unknown iterations courses = [] while True: course = input("Enter course (or 'done'): ") if course.lower() == 'done': break courses.append(course) # For loop - known iterations for i in range(1, 6): print(f"Count: {i}") # Nested loops for row in range(3): for col in range(3): print(f"({row},{col})", end=" ") print()

Control structures are the logic engine of your program - they determine what happens and when!
