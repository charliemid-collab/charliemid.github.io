Common Error Types
ValueError
Invalid value for operation
TypeError
Wrong data type used
ZeroDivisionError
Division by zero
IndexError
List index out of range
Try-Except Structure
# Safe input handling try: gpa = float(input("Enter GPA: ")) if gpa < 0 or gpa > 4.0: raise ValueError("GPA must be between 0.0 and 4.0") classification = calculate_classification(gpa) except ValueError as e: print(f"Invalid input: {e}") gpa = 0.0 # Default value except TypeError as e: print(f"Type error: {e}") except Exception as e: print(f"Unexpected error: {e}")
