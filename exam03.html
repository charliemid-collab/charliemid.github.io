Inheritance and Polymorphism

Part (a) - Inheritance Hierarchy (15 marks)

Create an inheritance hierarchy for university personnel:

  • Base class Person with name, id, email attributes
  • Derived class Student (from Person) with gpa, courses attributes
  • Derived class Instructor (from Person) with department, salary attributes
  • Derived class Staff (from Person) with position, office attributes
  • Each class should have appropriate __init__ and __str__ methods

Include proper constructor chaining using super()

Part (b) - Method Overriding (8 marks)

Implement method overriding in the inheritance hierarchy:

  • Base class method get_role() that returns "Person"
  • Override get_role() in each derived class to return appropriate role
  • Base class method get_details() that returns basic info
  • Override get_details() in derived classes to include specific attributes

Demonstrate polymorphism by calling these methods on different object types

Part (c) - Polymorphism in Action (7 marks)

Create a polymorphic demonstration:

  • Create a list containing different types of Person objects
  • Iterate through the list and call get_role() on each object
  • Iterate through the list and call get_details() on each object
  • Demonstrate that the correct method is called based on the actual object type
